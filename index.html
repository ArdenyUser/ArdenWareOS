Arden Ware OS (Thorium)

Simple non-gui OS.

 

Welcome to Arden Ware OS brought to you by Ardeny Foundation employee Avery Stafford and built off of Thor OS. This project is under development. Please compile via wichtounet's tutorial to compile Thor OS. Kernal is about the same as Thors. Very similar to Thor currently. Expect updates until on the page we say there are none. Please run with QEMU.
